## Filter plugin : Uuid

* Author: InterestingLab
* Homepage: https://interestinglab.github.io/waterdrop
* Version: 1.0.0

### Description

为原始数据集新增自增id字段

### Options

| name | type | required | default value |
| --- | --- | --- | --- |
| [target_field](#target_field-string) | string | no | uuid |

##### target_field [string]

自增id字段，若不配置默认为`uuid`
